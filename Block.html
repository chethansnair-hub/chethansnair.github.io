This was my first project using the Unity Engine. The purpose of this project
was to experiment with machine learning, a method used to train Artificial Intelligence.
Basically, the agent (blue cube) needs to hit the randomly positioned grey cube. Every time the agent touches the
grey cube, it receives a reward. Over time, the agent will learn to touch the grey cube.
If the agent falls off the platform, it receives a negative punishment. The entire concept of machine learning is
learning from trial and error.

This project was created for the purpose of presenting in my school science fair. It is safe to say it worked
