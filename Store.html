This was a project done by me and my java tutor, my granduncle.
We did this project so I could learn CRUD operations and file writing and saving.
Basically, this is a text-only simulation where you run a store. You have the options to insert,
Display, Search, Delete, Update, List Items, Buy items and display a bill, and Restock with an invoice.
